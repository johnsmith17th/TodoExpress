#TodoExpress
<p>This is a simple todo-list application on nodejs.</p>

###HOW TO USE
1. Install nodejs.
2. Download this project and unzip.
3. To run the application, the follow modules are also required:
>
* express
* jade
* mongodb
* mongoose

###JUST RUN IT
    node server.js
And it's all there: http://localhost:8080
